# push-button-lock-using-ESP-32
a project using push buttons, ESP 32 and LED'S 
<hr>
This project is used as a lock using three buttons and a ESP-32 micro controller and with two indicating LED'S.
some of the bugs i faced were while assigning global variables,freezing of ESP-32 after one atempt,confusion in button state (i.e. HIGH and LOW) and some small syntax errors missing terminators.
